# CubeMatrix

*CubeMatrix*是一款休闲益智3D小游戏，操作极其简单，滑动屏幕即可翻转方块。
游戏过程中你必须想方设法来推测每一个步骤，因为错误的步骤可能会导致方块无法前行。最终将方块垂直落入目标即可获胜。
相信这款游戏一定可以成为你们消遣休闲首选!


>- 纯代码实现3D效果
>- 流畅的动画效果
>- 安装包体积仅70KB
>- 地图格式简单易编写
>- 实现了简易VR模式
